NBA App 

Reuirements:
- You must have Nodejs installed

How To Run:

1. You have to start your IDE (I used Visual Studio Code).
2. Add this whole file.
3. Add a terminal
4. Run the following command: "npm run dev".
5. The terminal will write the follwing line: "Server On Port 4000".
6. Open your favorite browser.
7. Write "http://localhost:4000/" in a new tab.
8. Add the desired integer and enjoy the outcome.
